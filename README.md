# Practica1TAW
la estructura del proyecto es:
estudiante api
 └─ src
    └─ main
       ├─ java
       │  └─ com.example.estudianteapi
       │     ├─ controller
       │     ├─ dto
       │     ├─ entity
       │     ├─ repository
       │     └─ service
       └─ resources
          ├─ application.properties

destro de cada packete de com.exmple.estudianteapi hay un archivo java, todo esto funciona con los comandos
mvn clean install
mvn spring-boot:run
